better_robot.io.ir#

Intermediate representation — parser output, build_model() input.

All parsers (URDF, MJCF, programmatic builder) emit an IRModel. A single build_model() factory consumes the IR and produces a frozen Model. The IR is flat and ordering-unconstrained; topology/idx_q/idx_v assignment is build_model’s job.

See docs/concepts/parsers_and_ir.md §2.
