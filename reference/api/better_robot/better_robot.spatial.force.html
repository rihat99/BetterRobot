better_robot.spatial.force#

Force — 6D spatial wrench value type (dual of Motion).

Stored as (..., 6) tensor [fx, fy, fz, tx, ty, tz].

See docs/concepts/lie_and_spatial.md §7.

abstract cross_motion(other) → Motion[source]#

Not a standard spatial operation — use :meth:Motion.cross_force.

Force × Motion is not part of the spatial-algebra primitives. The dual of :meth:Motion.cross_force is what callers usually want.

se3_action(T) → better_robot.spatial.force.Force[source]#

Apply an SE3 transform to the wrench (dual adjoint action).

T may be a raw (..., 7) tensor or an

Class:

~better_robot.lie.types.SE3 value (its .tensor is unwrapped).

F_new = Ad(T)^{-T} * F = Ad^*(T) * F; for spatial forces Ad^*(T) = Ad(T^{-1})^T.
